# youtube-downloader
A downloader for Youtube audios and videos.
# Dependecies:
- Flask
- yt-dlp
- Python
- Git
# How to use:
Clone this repository using:
```
git clone https://github.com/FriskUT/youtube-downloader.git
```
Then run the server using:
```
python app.py
```
